Debug This Code

Find and fix the error in the following Python code snippet. This challenge will test your understanding of Python syntax and common pitfalls.

        def calculate_average(numbers):
            total = 0
            count = 0
            for num in numbers:
                total += num
                count += 1
            average = total / count
            return average
        
        numbers = [10, 20, 30, 40, '50']
        result = calculate_average(numbers)
        print(f"The average is: {result}")
numbers = [10, 20, 30, 40, '50']

What is the error in this code?

Correct! The list contains a string '50' which can't be added to numbers, causing a TypeError.

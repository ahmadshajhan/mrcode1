Python Basics

Let's start with the fundamental building blocks of Python programming.

Variables and Data Types

Python is dynamically typed, meaning you don't need to declare variable types.

# Basic data types
name = "Alice"          # String
age = 30               # Integer
price = 19.99           # Float
is_student = True       # Boolean
fruits = ["apple", "banana", "cherry"]  # List

Control Flow

Python uses indentation to define code blocks (typically 4 spaces).

if age < 18:
    print("You're a minor")
elif age >= 18 and age < 65:
    print("You're an adult")
else:
    print("You're a senior")
# For loop
for fruit in fruits:
    print(fruit)
    
# While loop
count = 0
while count < 5:
    print(count)
    count += 1
# Match case (Python 3.10+)
match status_code:
    case 200:
        print("Success")
    case 404:
        print("Not found")
    case 500:
        print("Server error")
    case _:
        print("Unknown status")

Functions

Functions are defined using the def keyword.

def greet(name):
    return f"Hello, {name}!"

# Call the function
message = greet("Bob")
print(message)  # Output: Hello, Bob!

Intermediate Python

Now that you know the basics, let's explore more powerful Python features.

Lists, Tuples, Dictionaries, and Sets

# List (mutable, ordered)
numbers = [1, 2, 3]
numbers.append(4)  # [1, 2, 3, 4]

# Tuple (immutable, ordered)
coordinates = (10.0, 20.0)

# Dictionary (mutable, key-value pairs)
person = {"name": "Alice", "age": 30}

# Set (mutable, unordered, unique elements)
unique_numbers = {1, 2, 3, 3}  # {1, 2, 3}

List Comprehensions

A concise way to create lists.

# Traditional approach
squares = []
for x in range(10):
    squares.append(x**2)

# With list comprehension
squares = [x**2 for x in range(10)]

Error Handling

Use try-except blocks to handle exceptions gracefully.

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
except Exception as e:
    print(f"An error occurred: {e}")
finally:
    print("This always executes")

Advanced Python Concepts

These features make Python powerful and flexible for complex applications.

Object-Oriented Programming

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hi, I'm {self.name} and I'm {self.age} years old."

# Create an instance
alice = Person("Alice", 30)
print(alice.greet())

Decorators

Functions that modify the behavior of other functions.

def timer_decorator(func):
    import time
    
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()
        print(f"Function {func.__name__} took {end_time - start_time:.4f} seconds")
        return result
    
    return wrapper

@timer_decorator
def slow_function():
    import time
    time.sleep(2)

slow_function()

Generators

Efficient way to handle large datasets by yielding items one at a time.

def fibonacci(limit):
    a, b = 0, 1
    while a < limit:
        yield a
        a, b = b, a + b

# Usage
for num in fibonacci(100):
    print(num)

Python for Web Development

Python is widely used for building web applications with frameworks like Django and Flask.

Flask Example

# Install Flask first: pip install flask
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Welcome to my Flask app!"

@app.route('/hello/<name>')
def hello(name):
    return f"Hello, {name}!"

if __name__ == '__main__':
    app.run(debug=True)

Django Example

Django is a more batteries-included framework.

# Install Django: pip install django
# Create project: django-admin startproject myproject
# Create app: python manage.py startapp myapp

# myapp/views.py
from django.http import HttpResponse

def index(request):
    return HttpResponse("Hello, Django!")

# myproject/urls.py
from django.urls import path
from myapp import views

urlpatterns = [
    path('', views.index, name='index'),
]

FastAPI Example

Modern, fast (high-performance) web framework for building APIs.

# Install FastAPI: pip install fastapi uvicorn
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Welcome to FastAPI"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: str = None):
    return {"item_id": item_id, "q": q}

Python for Data Science

Python is the leading language for data analysis and machine learning.

NumPy Basics

Fundamental package for numerical computing.

import numpy as np

# Create arrays
arr = np.array([1, 2, 3, 4, 5])
matrix = np.array([[1, 2], [3, 4]])

# Basic operations
print(arr * 2)      # [2, 4, 6, 8, 10]
print(matrix.T)    # Transpose
print(np.dot(matrix, matrix))  # Matrix multiplication

Pandas Basics

Powerful data manipulation and analysis library.

import pandas as pd

# Create DataFrame
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['New York', 'London', 'Paris']
}
df = pd.DataFrame(data)

# Basic operations
print(df.head())       # First 5 rows
print(df.describe())   # Statistics
print(df[df['Age'] > 28])  # Filtering

plt.plot(x, y)
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.title('Simple Plot')
plt.show()
